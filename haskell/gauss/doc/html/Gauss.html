gaussianElimination
:: [[Double]]Coefficient matrix A
-> [Double]Right hand side column vector b
-> [Double]Returns solution vector x

Solve the equation Ax = b using Gaussian elimination.

For example, if 

          4.0  -2.0   1.0
     A =  2.0   4.0  -2.0
          1.0  -2.0   4.0

and b = [11.0, - 16.0, 17.0]' then, the expected solution vector x is [1.0, -2.0, 3.0]'.

gaussianEliminationWithScaledRowPivoting
:: [[Double]]Coefficient matrix A
-> [Double]Right hand side column vector b
-> [Double]Returns solution vector x

Solve the equation Ax = b using Gaussian elimination with scaled row pivoting.

For example, if 

           2.0  -2.0   6.0
      A = -2.0   4.0   3.0
          -1.0  -8.0   0.0

and b = [61.0, 0.0, -1.0]' then, the expected solution vector x is [1.0, -1.0, 2.0]'.
